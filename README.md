# 🔀 Proxy Redirector

برنامج بايثون لإدارة البروكسيات تلقائياً: فحص دوري، ترتيب ذكي، ونفق SOCKS5 محلي مع تبديل تلقائي (failover).

## المميزات

- ✅ فحص تلقائي لجميع البروكسيات كل دقيقة (قابل للتعديل)
- ✅ دعم **SOCKS5**, **SOCKS4**, **HTTPS**
- ✅ ترتيب ذكي بالنقاط (سرعة، موثوقية، حداثة)
- ✅ نفق **SOCKS5 محلي** (`127.0.0.1:1080`)
- ✅ **تبديل تلقائي** (Failover) عند فشل البروكسي
- ✅ Dashboard مباشر في الكونسول
- ✅ دعم البروكسيات بمصادقة (username/password)

## التثبيت

```bash
# تثبيت المكتبات
pip install -r requirements.txt
```

## الاستخدام

### 1. إعداد البروكسيات

حرّر ملف `proxies.json` وأضف بروكسياتك:

```json
{
  "proxies": [
    {
      "id": "proxy_001",
      "ip": "103.152.112.186",
      "port": 8080,
      "type": "socks5",
      "username": null,
      "password": null
    }
  ]
}
```

**الأنواع المدعومة:** `socks5`, `socks4`, `https`

### 2. تشغيل البرنامج

```bash
python main.py
```

### 3. استخدام البروكسي المحلي

في **ProxyFire** أو أي برنامج:
- **النوع:** SOCKS5
- **العنوان:** 127.0.0.1
- **البورت:** 1080

### 4. اختبار يدوي

```bash
curl --socks5 127.0.0.1:1080 http://httpbin.org/ip
```

## الإعدادات

جميع الإعدادات في ملف `config.py`:

| الإعداد | القيمة الافتراضية | الوصف |
|---------|-------------------|-------|
| `CHECK_INTERVAL_SECONDS` | `60` | فترة الفحص بالثواني |
| `CHECK_TIMEOUT_SECONDS` | `10` | مهلة فحص كل بروكسي |
| `MAX_CONCURRENT_CHECKS` | `20` | أقصى فحوصات متزامنة |
| `LOCAL_PORT` | `1080` | بورت SOCKS5 المحلي |
| `FAILOVER_MAX_RETRIES` | `3` | محاولات قبل التبديل |


## خوارزمية الترتيب

كل بروكسي يحصل على نقاط (score):

| العامل | الوزن |
|--------|-------|
| يعمل حالياً | +50 |
| سرعة الاستجابة | 0 إلى +25 |
| حداثة آخر نجاح | 0 إلى +15 |
| نسبة النجاح الكلية | 0 إلى +10 |
| فشل متتالي | -5 لكل مرة |

البروكسي ذو أعلى نقاط يُستخدم أولاً.
